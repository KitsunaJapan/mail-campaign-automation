1
このリポジトリをGitHubにプッシュ
ダウンロードしたファイル一式（server.js、package.json、public/index.html）をGitHubの新しいリポジトリにプッシュします。
.envは 絶対にプッシュしないでください（.gitignoreに記載済み）。
2
Render で新しいWeb Serviceを作成
render.com →
「New +」→「Web Service」→ GitHubリポジトリを接続します。
3
ビルド設定を入力
Runtime: Node
Build Command: npm install
Start Command: node server.js
Instance Type: Free（無料プランで動作します）
4
環境変数を設定（最重要）
Renderダッシュボード →「Environment」タブ → 以下を追加：

ANTHROPIC_API_KEY = sk-ant-xxxxx
GOOGLE_API_KEY = AIzaSyxxxxx

ここで設定したキーはコードに含まれず、安全に管理されます。
5
「Create Web Service」でデプロイ完了
数分でビルドが完了し、https://your-app.onrender.com のURLが発行されます。
無料プランは15分で自動スリープしますが、アクセスすると自動で起動します。
常時起動が必要な場合は有料プラン（$7/月〜）をご検討ください。
